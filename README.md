# Clinic Manager(PyQt5 + Sqlite)

Intro: this app is for super tiny scale single doctor's clinic PC.

### on Linux

```bash
chmod +x install.sh  lang_pack_gen.sh  run.sh
./install.sh
./lang_pack_gen.sh
./run.sh
```

### on Windows

double click `configure.bat` and `install.bat lang_pack_gen.sh`
